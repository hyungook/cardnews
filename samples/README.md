# 📋 구글 스프레드시트 샘플 사용법

## 방법 1: CSV 파일 가져오기 (가장 간단)

1. [Google Sheets](https://sheets.google.com)에서 **새 스프레드시트**를 만듭니다.
2. 시트 이름을 **"데이터"**로 변경합니다. (하단 탭 더블클릭)
3. 메뉴에서 **파일 → 가져오기**를 클릭합니다.
4. **업로드** 탭에서 `카드뉴스_데이터_샘플.csv` 파일을 선택합니다.
5. "가져오기 위치"를 **"현재 시트 바꾸기"**로 선택하고 가져옵니다.
6. 하단에서 **+** 버튼을 눌러 새 시트 탭을 추가하고 이름을 **"히스토리"**로 변경합니다.
7. 히스토리 탭의 1행에 아래 헤더를 입력합니다:

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| ID | 생성일시 | 총 건수 | 성공 | 오류 | 파일목록 | 오류목록 |


## 방법 2: 직접 입력

### "데이터" 시트 (첫 번째 탭)

| | A | B | C | D | E | F | G | H | I (선택) | J (선택) |
|---|---|---|---|---|---|---|---|---|---|---|
| **1행 (헤더)** | 영화제목 | 기본문구 | 추가문구 | 뱃지1 | 뱃지2 | 뱃지3 | 뱃지4 | 카피라이트 | 배경파일명(수동) | 로고파일명(수동) |
| **2행 (예시)** | 범죄도시4 | 올 여름↵최고의 액션 | 제62회 대종상 수상 | 무료 | UHD | | | © 2024 ABO | | |

> ↵ = 줄바꿈 (Ctrl+Enter 또는 Mac에서 ⌘+Enter)

### "히스토리" 시트 (두 번째 탭)

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| ID | 생성일시 | 총 건수 | 성공 | 오류 | 파일목록 | 오류목록 |

> 이 탭은 비워두세요. 카드뉴스 생성 시 자동으로 기록됩니다.


## 컬럼 설명

| 컬럼 | 필수 | 설명 |
|------|------|------|
| 영화제목 | ✅ | 영화/드라마 제목. 배경/로고 파일명 자동 생성에 사용됨 |
| 기본문구 | ✅ | 홍보 문구 (최대 3줄) |
| 추가문구 | | 수상내역 등. 비워두면 숨김 처리 |
| 뱃지1~4 | | 20종 중 선택 (최대 4개). 비워두면 뱃지 숨김 |
| 카피라이트 | | 저작권 표기. 비워두면 숨김 처리 |
| 배경파일명(수동) | | 파일명이 규칙과 다를 때만 입력 |
| 로고파일명(수동) | | 파일명이 규칙과 다를 때만 입력 |


## 파일명 자동 생성 규칙

영화제목만 입력하면 배경/로고 파일명이 자동으로 결정됩니다:

- **배경**: `{영화제목}.jpg` → 예: `범죄도시4.jpg`
- **로고**: `LI_{영화제목}.png` → 예: `LI_범죄도시4.png`

파일명이 이 규칙과 다른 경우에만 I, J열에 직접 입력하세요.


## 중요: 서비스 계정에 공유하기

스프레드시트를 만든 후, 반드시 서비스 계정 이메일에 **편집자** 권한으로 공유해야 합니다.

1. 스프레드시트 오른쪽 위 **"공유"** 버튼 클릭
2. 서비스 계정 이메일 입력 (예: `cardnews@프로젝트명.iam.gserviceaccount.com`)
3. 권한을 **"편집자"**로 설정
4. **"보내기"** 클릭
